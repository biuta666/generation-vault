# Pricing Strategy

> 内部文档。不公开。

---

## 核心理念

买断制，不订阅。因为：
- 本地运行，零服务器成本
- 订阅会让AI创作者反感（"你又没服务器"）
- 符合本地工具的用户预期（Eagle $29.95, Directory Opus $59）

## 定价逻辑

为什么不是 $9:
- 太便宜 → 用户觉得是玩具
- 不认真对待

为什么不是 $99:
- 没有品牌溢价
- 没有口碑积累

为什么是 $29-49:
- Eagle 已经教育了市场 "本地工具 ≈ $30"
- 设计师/创作者接受这个区间
- 比Eagle多一个Pro版($49)覆盖Power User

---

## 验证标准

在定最终价格之前，先验证：
- 有没有5个人说 "这个值30美元"
- 有没有人因为没有它而觉得工作流变麻烦
- GitHub Issues 的自然反馈

出现第一个付费意愿的人 → 价格就定了。
